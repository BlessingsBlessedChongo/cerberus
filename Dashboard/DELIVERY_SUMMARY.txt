Authentication:

  Body: { "username": "admin", "password": "pass" }
  Response: { "token": "...", "username": "...", "user_id": 1 }

Latest Metrics:
  GET /api/metrics/latest/?page_size=5
  Auth Header: Authorization: Token {token}
  Response: { "results": [
    { "metric_type": "CPU_Usage", "value": 45.2 },
    { "metric_type": "Available_RAM", "value": 8192 },
    { "metric_type": "Disk_Free_C", "value": 256.5 }
  ]}

CPU History:
  GET /api/metrics/history/?hours=24&type=CPU_Usage
  Auth Header: Authorization: Token {token}
  Response: { "results": [
    { "timestamp": "2024-01-15T14:30:00Z", "value": 45.2 },
    { "timestamp": "2024-01-15T14:29:00Z", "value": 42.1 }
  ]}

RAM History:
  GET /api/metrics/history/?hours=24&type=Available_RAM
  Auth Header: Authorization: Token {token}
  Response: { "results": [...] }

Alerts:

  Auth Header: Authorization: Token {token}

================================================================================
   CUSTOMIZATION GUIDE
================================================================================

Change API URL:
  Edit: Dashboard/src/pages/Login.jsx, line 7
  And:  Dashboard/src/pages/Dashboard.jsx, line 18
  Find: const API_BASE = 'http://localhost:8000/api'
  Update to your backend URL

Change Theme Colors:
  Edit: Dashboard/src/index.css, lines 1-19
  Update :root CSS variables

Change Polling Interval:
  Edit: Dashboard/src/pages/Dashboard.jsx, line 165
  Change: setInterval(loadLatest, 10000)
  (10000 ms = 10 seconds)

Add More Metrics:
  1. Copy card div in Dashboard.jsx
  2. Add useState for new metric
  3. Update loadLatest() to fetch it
  4. Display in card
